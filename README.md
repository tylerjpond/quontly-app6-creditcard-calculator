# Quontly Credit Card Payoff Calculator

This project is a SvelteKit implementation of Quontly's credit card payoff calculator.

## Stack

- SvelteKit
- Cloudflare adapter
- Tailwind CSS v4
- DaisyUI
- TypeScript
- Zod

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run check` runs `svelte-check`.
- `npm run lint` runs ESLint.
- `npm run preview` previews the production build locally.

## Notes

- The calculator logic lives in `src/lib/creditCard.ts`.
- Marketing, FAQ, and legal content lives in `src/content/site.ts`.
- The SvelteKit migration plan is documented in `docs/sveltekit-refactor-plan.md`.
